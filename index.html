集美大学JMU校园网使用攻略

发表于 2023-11-10 更新于 2026-03-16 本文字数： 2k 阅读时长 ≈ 2 分钟

你说得对，但是检测出非实名上网行为，系统将自动降速10分钟，请立即停止违规上网！根据《中华人民共和国网络安全法》、公安部第82号令等规定，为保障校园网的安全，接入校园网必须进行实名认证。账号持有人须对账号身份信息的真实性、安全性负责。

所谓上有政策，下有对策。本篇教程将会讲述如何在🐔🧱的校园网继续使用路由器进行网上冲浪。

文章使用设备和环境

  • 以 Hanwckf 的 Immortalwrt-mt798x 为例
  • 本人使用的是 CMCC RAX3000M/Redmi AX6000, 其实都差不多

多设备检测

通过众多的相关测试可以发现，校园网使用的检测机制：

  • ❌ User-Agent 检测
  • ❌ IPID/TTL/时钟偏移检测
    即使针对上述策略优化后，路由器仍会被检测。排除法合理推测 学校可能正在使用 DPI（深度包检测）技术。不过在尝试后发现只要桥接一下就可以绕过了

解决方案：无线桥接

  1. 物理准备
    拔掉路由器连接在墙上网口的网线

  2. 配置无线
    访问路由器管理页面：
    网络 → 无线 → 选择 apclix (5GHz WiFi) → 扫描


  3. 连接校园网
    选择信号最佳的 JMU-STU 并保存

  4. 验证接口
    网络 → 接口 → 确认 WAN/WAN6 使用 apclix0

  5. 完成认证
    浏览器访问 10.8.2.2 进行认证
    ⚡ 实测效果：

    • Speedtest：60-70Mbps100Mbps左右（可能是设备升级？今年网速变快了）
    • 游戏延迟：无畏契约/CS/王者荣耀均在正常水平

断网期间认证上网

众所周知，我们🐔🧱是分上网区域的。有需要按照断网时间断网的宿舍区域，也有 不会断网的办公区域 。

经过尝试，发现使用 静态 IP 竟然可以达到预期的效果。

操作步骤

  1. （最难的一步）获取关键信息
    准备一个办公区域的内网IP（如 172.AA.XX.XX）

    上图是网络自助里的一个小模块，实验课有机会去上网留下的记录。这里推荐尝试能不能拿到尚大楼的内网IP段好像哪怕上面写的是办公区域，也不一定能行？但是我知道一个肯定能上网的IP段，这里不直接写。不过其实只需要寻找IP段里的AA部分，后面XX.XX随便填个小于255的数字实测都能上网的。这个AA又都是30以内的数字。你可以一个一个试？

  2. 配置静态IP
    网络 → 接口 → 编辑 WAN 接口：

    • 协议：静态地址（原为DHCP客户端）

    • 参数设置：
    • 设置完之后去高级设置里加几个DNS服务器



  3. 断网时段认证
    保存后访问 10.8.2.2 即可正常认证

无路由器解决方案

▫ Windows

  1. 连接 JMU-STU → 打开网络设置

  2. 更改适配器选项 → 右击WLAN → 属性

  3. 双击 IPv4协议 → 填写办公区静态IP参数



▫ iOS/iPadOS

  1. 设置 → 无线局域网 → JMU-STU旁边的设置小按钮，配置 IPv4
  2. 配置IP → 选择手动
  3. 填写静态IP并配置DNS（如 8.8.8.8）

▫ Android

因系统碎片化严重，请参照上述原理自行设置

注意事项

  1. 稳定性问题
    • 在单次连接后可能随着时间推移越来越卡，推荐定时重启路由器并重新连接。有些时候设置静态IP后可能进不去认证界面，需要反复尝试。
  2. 无法连接校内内网网站(2023-03-09更新)
    • 字面意思，合理怀疑是需要找找学校校内的DNS服务器，有机会再看看。
    • 更新：可能是因为 DNS Rebinding protection, 认证界面的稳定性问题可能也是因为这个。
  3. 个人推荐
    • 就目前来看，如果你没有长期下载大量大型文件的需求（比如我以前很爱动不动就下载几十上百G的源码和游戏），以及你现在是低年级，来日方长的话，我推荐你去购买相关的5G CPE（一千多？）和流量卡套餐。
  4. 其他问题
    • 在选择使用的静态IP下会不会刚好顶掉一个真实存在的机子？这样干万一碰到了会不会有什么影响？but I don’t know.

2026-03-09 更新

解决校内访问问题方案：

如果你是在用OpenWRT，将 网络 -> DHCP/DNS -> 重绑定保护 关闭。然后将上方WWAN中配置的 “使用自定义DNS服务器“里的DNS地址改为校内的DNS服务器地址。
如果你是单纯在自己的手机电脑等设备上连接，直接将DNS修改为校内的DNS服务器地址(我没试)应该也可以。

校内的DNS服务器地址：
连接JMU_STU后通过ipconfig /all直接获取。
